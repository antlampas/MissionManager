distribuiti con licenza **Creative Commons Attribution-ShareAlike 4.0
International**.

```text

```

## Attribuzione

Attribuzione consigliata, salvo indicazioni diverse nei singoli materiali:

```text
Sistema ACL, Francesco, licenza Creative Commons Attribution-ShareAlike 4.0 International.
```

Quando condividi copie o adattamenti del materiale:

- conserva questa nota di licenza;
- attribuisci il progetto e, quando disponibile, collega la fonte originale;
- indica in modo ragionevole se hai modificato il materiale;
- distribuisci eventuali adattamenti con `CC BY-SA 4.0` o con una licenza
  compatibile ShareAlike.

## Ambito

La licenza copre i contenuti del repository, inclusi documenti Markdown e
diagrammi PlantUML, salvo file o sezioni che dichiarino esplicitamente una
licenza diversa.

La licenza non modifica eventuali diritti su marchi, nomi di terzi, dati esterni
o materiali incorporati che siano soggetti a condizioni proprie.
